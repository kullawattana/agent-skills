# Flutter Development

## When to Use
- Create and customize Flutter widgets
- Manage state with Provider, Riverpod, or BLoC
- Connect to REST API or Firebase
- Develop UI/UX for mobile apps
- Handle navigation and routing

## Procedure
1. Plan app structure and required features
2. Create basic widgets and layout
3. Implement state management for data handling
4. Connect to APIs and manage data
5. Test and optimize performance
6. Prepare for build and deployment

## Assets
This skill includes:
- Ready-to-use widget templates
- Scripts for project setup
- State management code examples
- Flutter patterns reference guide
