# Flutter Solution Architect

## When to Use
- Design scalable folder structures for large Flutter projects
- Implement clean architecture and MVVM patterns
- Plan state management strategies for enterprise apps
- Create modular and maintainable code organization
- Design for scalability and team collaboration

## Procedure
1. Analyze project requirements and scale
2. Design folder structure and module organization
3. Define architecture patterns (Clean Architecture, MVVM)
4. Plan state management and dependency injection
5. Create data flow and layer separation
6. Document architecture decisions and guidelines
