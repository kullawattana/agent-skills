# API Testing

## When to Use
- Testing REST API endpoints
- Validating API responses and status codes
- Performing integration testing
- Automating API test suites

## Procedure
1. Define API endpoints and expected behaviors
2. Set up test environment and authentication
3. Create test cases for different scenarios
4. Execute requests using testing frameworks
5. Validate responses and error handling
6. Generate test reports and metrics

## Assets
This skill includes:
- Scripts for API testing automation
- Templates for test case creation
- Reference guides for API testing best practices
