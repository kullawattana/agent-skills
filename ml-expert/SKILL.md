# ML Expert

## When to Use
- Generating machine learning models for prediction tasks
- Performing data analytics and visualization
- Building predictive models from datasets
- Analyzing trends and patterns in data

## Procedure
1. Assess the data requirements and model type needed
2. Prepare and preprocess the dataset
3. Select appropriate ML algorithm
4. Train and validate the model
5. Generate analytics and insights
6. Provide model evaluation metrics

## Assets
This skill includes:
- Scripts for common ML tasks
- Templates for data preprocessing
- Reference documentation for best practices
