# Data Visualization

## When to Use
- Creating charts and graphs from datasets
- Visualizing data distributions and relationships
- Generating plots for reports and presentations
- Exploring data patterns through visualization

## Procedure
1. Analyze the data structure and requirements
2. Select appropriate visualization type (bar, line, scatter, etc.)
3. Prepare data for plotting
4. Generate the visualization using matplotlib/seaborn
5. Customize appearance and add labels
6. Save or display the final chart

## Assets
This skill includes:
- Scripts for common chart types
- Templates for data preparation
- Reference guides for visualization best practices
